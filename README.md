# 🛡️ Astra Hardening

Скрипт для автоматической настройки безопасности Astra Linux Common Edition после установки.

---

## 📦 Что делает

- Обновляет систему
- Настраивает файрвол (ufw) — разрешён только SSH
- Отключает ненужные службы (cups, bluetooth, avahi-daemon)
- Настраивает политику паролей (минимальная длина 12 символов)
- Включает автоматические обновления безопасности
- Включает аудит (auditd)
- Настраивает SSH (запрет root-входа с паролем, отключение парольной аутентификации)

---

## 🚀 Запуск

```bash
chmod +x hardening.sh
sudo ./hardening.sh
```

---

## 📁 Логи
Все действия записываются в файл /var/log/hardening.log.

---

## ⚠️ Внимание
Перед запуском убедитесь, что у вас есть SSH-доступ по ключу, чтобы не потерять доступ к системе.
